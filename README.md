# e-plantShopping

## Project Overview
Paradise Nursery - A modern e-commerce plant shopping application built with React and Redux.

## Features
- Browse plants by categories (Air Purifying, Aromatic Fragrant, Insect Repellent, Medicinal, Low Maintenance)
- Add plants to shopping cart
- Real-time cart quantity display
- Increment/Decrement product quantities
- Remove items from cart
- Calculate total costs
- Responsive design for all screen sizes

## Technologies Used
- React 18.2.0
- Redux Toolkit 2.2.3
- Vite 5.4.21
- React-Redux 9.1.1
